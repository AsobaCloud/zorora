# SEP-016: Guided Query Refinement

**Status:** Open
**Created:** 2026-03-02
**Project:** zorora

## Summary
Add a guided query refinement step that detects missing dimensions (time bounds, scope, analysis type) and prompts users to fill gaps before deep research starts, similar to ChatGPT Deep Research UX

## Motivation
Underspecified queries produce vague synthesis; no date anchoring causes hallucination of future events in follow-up chat; the tool needs structured input for energy, regulatory, and intelligence research domains

## Proposed Change
New query_refiner module, refinement API endpoint, interactive refinement panel in UI with option pills, date anchoring and anti-hallucination constraints in all LLM prompts

## Acceptance Criteria
Refinement endpoint returns structured gaps for vague queries; UI shows refinement panel with clickable options; synthesis and chat prompts include today date; follow-ups no longer fabricate future events
