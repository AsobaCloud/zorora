# SEP-042: SAPP & Eskom Data Parsers

## Summary
Parse the downloaded Southern Africa energy datasets (SAPP DAM prices, Eskom operational data, Eskom tariffs, GCCA GeoPackage) into structured data accessible by the platform.

## Motivation
Zorora has no Southern Africa power market data integration despite having downloaded datasets covering SAPP day-ahead market prices (3 nodes, 15 months hourly), Eskom demand/generation/station build-up, Eskom 2025/26 tariff schedules (26 tariff types with TOU rates), and GCCA grid infrastructure (159 MTS substations with polygon zones). These datasets are the foundation for BESS site selection, feasibility analysis, and diligence.

## Change Description

### New Clients
- `tools/market/sapp_client.py` — Parse SAPP DAM xlsx files (RSAN, RSAS, ZIM) into hourly price series by node. Fields: area, delivery_day, delivery_hour, price_usd, price_zar.
- `tools/market/eskom_client.py` — Parse Eskom CSVs: System_hourly_actual_and_forecasted_demand.csv, Hourly_Generation.csv, Station_Build_Up.csv. Return structured time series.
- `tools/regulatory/eskom_tariff_client.py` — Parse Eskom-tariffs-1-April-2025-ver-2.xlsm into structured TOU rates by tariff type, transmission zone, voltage level, season (high/low demand), period (peak/standard/off-peak).
- `tools/imaging/gcca_client.py` — Parse GCCA 2025 GIS/AREAS_GCCA2025.gpkg (GeoPackage with 3 layers: MTS_ZONES 159 substations, SUPPLY_AREA 10 regions, LOCAL_AREA 34 areas) into GeoJSON feature collections.

### Storage
- Extend `market_data.db` or create new store for SAPP price series
- Eskom tariff data as structured lookup (tariff → zone → voltage → season → period → rate)
- GCCA GeoJSON served from parsed GeoPackage (cache as JSON files or in imaging store)

### Configuration
- Add SAPP, ESKOM, GCCA sections to config.py with file paths and defaults

## Dependencies
None — this is foundational.

## Success Criteria
- SAPP DAM prices queryable by node, date range, hour
- Eskom demand/generation/station build-up queryable by date range
- Eskom tariff rates retrievable by tariff type, zone, voltage, season
- GCCA MTS zones, supply areas, local areas available as GeoJSON
- All parsers have unit tests covering actual data file formats
