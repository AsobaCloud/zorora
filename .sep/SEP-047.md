# SEP-047: Web UI market data refresh

**Status:** Open
**Created:** 2026-03-19
**Project:** zorora

## Summary
Start market/regulatory/alert background refresh threads from web_main.py

## Motivation
The refresh threads only start from main.py (REPL mode). web_main.py does not start them, so all market data including SAPP/Eskom never auto-loads or refreshes in web UI mode.

## Proposed Change
Add refresh thread startup calls to web_main.py before app.run()

## Acceptance Criteria
SAPP/Eskom data auto-loads on web UI startup and refreshes daily
