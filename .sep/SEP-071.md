# SEP-071: Prioritize newsroom sources in deep research ranking

**Status:** Open
**Created:** 2026-03-28
**Project:** zorora

## Summary
Boost newsroom (Ona curated articles) credibility and relevance scoring so curated energy/market content outranks generic Brave web search results in deep research aggregation

## Motivation
The newsroom API provides editorially curated articles on energy, geopolitics, and electricity trade — exactly the domains Zorora targets. Currently newsroom sources get credibility 0.75 (Tier 4), the same as major news outlets (Reuters 0.65-0.70) but below government sources (0.80). For energy domain queries, curated newsroom content is more relevant than generic Brave results which return SEO-optimized pages. Newsroom should be the primary evidence backbone for domain-specific research.

## Proposed Change
Increase newsroom base credibility to 0.85 for energy/market domain queries. Add a domain-relevance boost in the reranker: when query keywords match newsroom article topic_tags, apply a +0.10 relevance bonus. Ensure newsroom sources appear in the top 10 ranked sources when topic-relevant. Optionally increase newsroom max_results to fetch more curated content.

## Acceptance Criteria
Newsroom sources with matching topic tags rank in top 10 for domain-relevant queries. Newsroom credibility is 0.85 for energy domain queries. Curated articles outrank generic web results for energy/market/geopolitics topics.
