# SEP-041: Global View filter discovery and date range

**Status:** Open
**Created:** 2026-03-17
**Project:** zorora

## Summary
Replace free-text topic/source filters with data-driven multi-selects, expand cache window from 7 to 90 days, and increase tag visibility in tooltips and article rows

## Motivation
Users cannot discover available topic tags or sources without guessing; date filters beyond 7 days return blank results because the cache only stores a rolling 7-day window; tooltip and article row tag truncation hides available metadata

## Proposed Change
Widen newsroom cache window to 90 days, add /api/news-intel/facets endpoint to return unique topic_tags and sources, replace free-text inputs with multi-select dropdowns populated from facets, expand tag slice limits in tooltips and article table

## Acceptance Criteria
Topic and source filters populated from actual data; date range queries up to 90 days return results; tooltips show all topics; article rows show more tags
