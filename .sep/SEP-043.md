# SEP-043: Rename Imaging to Discovery + Grid Infrastructure Layers

## Summary
Rename the Imaging section to "Discovery" and add GCCA grid infrastructure layers (MTS substation zones, supply area boundaries, DAM price zones) to the map with color-by controls and BESS-relevant popups.

## Motivation
The Imaging view is the primary map-based asset discovery interface. Adding grid infrastructure layers enables investors to identify BESS-favorable geographies by overlaying substation zones with DAM pricing data. The rename to "Discovery" better reflects the section's purpose as pipeline/watchlist panels move to the new Scouting section (SEP-044).

## Change Description

### UI Changes
- Rename "Imaging" tab to "Discovery" throughout (navigation, code references, docs)
- Add "Grid Infrastructure" filter group with toggles:
  - MTS Substation Zones (159 polygons)
  - Supply Area Boundaries (10 polygons)
  - DAM Price Zones (RSAN / RSAS boundary)
- Add "Color zones by" dropdown: Arbitrage Spread | Avg DAM Price | RE Penetration
- MTS zone click popup showing: substation name, supply area, local area, DAM node, avg peak/off-peak prices, arbitrage spread, RE assets in zone
- Popup buttons: [Score BESS] [+ Scouting]

### API Endpoints
- `GET /api/discovery/gcca/mts-zones` — GeoJSON of MTS substation zones
- `GET /api/discovery/gcca/supply-areas` — GeoJSON of supply area boundaries
- `GET /api/discovery/dam-zones` — GeoJSON boundary between RSAN and RSAS
- `GET /api/discovery/zone-metrics/<substation>` — DAM price stats + RE asset count for a zone

### Map Rendering
- MTS zones as Leaflet polygon layer with configurable fill color based on selected metric
- Supply areas as thin border polygons (no fill)
- DAM zone boundary as dashed line
- Zone popup with formatted metrics and action buttons

## Dependencies
- SEP-042 (SAPP & Eskom data parsers — needed for DAM price metrics and zone coloring)

## Success Criteria
- "Discovery" label appears in navigation (Imaging references removed)
- MTS zones render on map when toggled on, colored by selected metric
- Clicking a zone shows popup with substation info and DAM price stats
- [+ Scouting] button adds zone to Scouting pipeline (SEP-044)
